# nbw-news
# Config
Create a file named `.env`  
Fill up the blank in the template  
DM me on WeChat if you donot you know how to find the urls
```
AUTH_DOMAIN = 
AUTH_PHASE1_URL = 
AUTH_PHASE3_URL = 
AUTH_PHASE4_URL = 
NBW_URL = 
PUSHPLUS_TOKEN = 
PUSHPLUS_TOPIC = 
```
# Setup
```bash
git clone https://github.com/ElysiaEgo/nbw-news.git
cd nbw-news
# install requirement
yarn install
# compile typescript
tsc
# run
node ./build/index.js
```
